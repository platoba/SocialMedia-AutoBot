# Changelog

## [5.0.0] - 2026-03-03

### Added
- 🚨 **危机监测模块** (`app/crisis_monitor.py`)
  - 实时监控品牌提及和负面评论
  - 负面情感分析和关键词检测
  - 病毒式传播预警（互动数阈值）
  - KOL/大V提及追踪
  - 负面激增检测（1小时内超过10条）
  - 四级危机等级评估（Critical/High/Medium/Low）
  - 危机响应建议生成
  - 24小时危机报告统计
- 📊 新增Telegram命令:
  - `/crisis` - 查看24小时危机监测报告
  - `/monitor <关键词>` - 启动实时品牌监控
- ✅ 完整测试覆盖 (`tests/test_crisis_monitor.py`)
  - 负面提及检测
  - 病毒式传播检测
  - KOL提及检测
  - 情感分析
  - 危机报告生成
  - 响应建议

### Technical Details
- 负面关键词库：14个中英文关键词
- 危机阈值配置：
  - 负面激增: 10条/小时
  - 病毒式传播: 1000互动
  - 情感骤降: -0.5分
- 情感分析范围: -1.0（极负面）到 1.0（极正面）
